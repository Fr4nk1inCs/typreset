# Fonts for Typreset

This branch contains fonts used as default in Typreset. For propietary fonts,
you can find the installation instructions in the `install.md` file in each
language folder.

The default font family for English is [Linux Libertine](https://linuxlibertine.sourceforge.net/Libertine-EN.html). It's bundled with Typst,
so you don't need to install it.

The default math font is [Libertinus](https://github.com/alerque/libertinus) Math, which you need to install manually.
For Arch-based Linux distributions, you can install it from `libertinus-font`
package in `extra` repository. You can also install it from
[its GitHub release page](https://github.com/alerque/libertinus/releases). This branch also contains [the font files](./libertinus) of Libertinus.

For language-specific fonts, see the corresponding language folder under `lang/`.
