# Install Simplified Chinese Fonts Used in Typreset

Some FREE [FounderType fonts](https://www.foundertype.com/) are used as
Typreset's default fonts for Simplified Chinese. If you want to use them, you
need to install them from there website.

|              Font Name             |        Usage       | Website |
|*----------------------------------*|*------------------*|*-------*|
|     方正书宋简体 FZShuSong-Z01S    |     base font      | [Link](https://www.foundertype.com/index.php/FontInfo/index/id/151) |
|       方正楷体简体 FZKai-Z03S      | emphasis (italic)  | [Link](https://www.foundertype.com/index.php/FontInfo/index/id/137) |
| 方正小标宋简体 FZXiaoBiaoSong-B05S |   strong (bold)    | [Link](https://www.foundertype.com/index.php/FontInfo/index/id/164) |

# 下载 Typreset 使用的中文字体

Typreset 使用了[方正字库](https://www.foundertype.com/)的部分字体作为默认的简体中文字体。如果你想使用这些字体，你需要从方正字库的网站上下载安装。

|              字体名称              |         用途       | 网站 |
|*----------------------------------*|*------------------*|*-------*|
|     方正书宋简体 FZShuSong-Z01S    |      基本字体      | [链接](https://www.foundertype.com/index.php/FontInfo/index/id/151) |
|       方正楷体简体 FZKai-Z03S      |    强调（斜体）    | [链接](https://www.foundertype.com/index.php/FontInfo/index/id/137) |
| 方正小标宋简体 FZXiaoBiaoSong-B05S |  着重强调（加粗）  | [链接](https://www.foundertype.com/index.php/FontInfo/index/id/164) |
